    Typora常用快捷键

    编辑功能

    新建 Ctrl + N

    复制为Markdown Ctrl + Shift + C

    粘贴为纯文本 Ctrl + Shift + V

    删除行（在表中） Ctrl + Shift +退格键

    查找 Ctrl + F

    查找下一个 F3 / Enter

    查找上一个 Shift + F3 / Shift + Enter

    替换 Ctrl + H

    段落功能

    标题1-6 Ctrl + 1/2/3/4/5/6

    插入表格 Ctrl + T

    插入数学公式 Ctrl + Shift + M

    插入代码块 Ctrl+Shift+K

    格式功能

    加粗 Ctrl + B

    下划线 Ctrl + U

    斜体 Ctrl+I

    代码 Ctrl+Shift+`
