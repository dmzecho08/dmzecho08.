    conda简单使用

    创建时间:2021-04-11 16:21
    阅读:
    1. 新建conda环境
    2. 激活conda环境
    3. 向环境中添加包
    4. 其他常用命令

    新建conda环境

    conda create -n 环境名
#如果有python版本要求，在末尾跟上，如python=2.7

    激活conda环境

    conda activate 环境名
#激活之后会自动进入该环境，为了使用方便，建议在 .bashrc文件中加入快捷方式

    向环境中添加包

    conda install 包名
#与pip install 效果一样

    其他常用命令

    conda info -e
#查看所有的虚拟环境

conda install tensorflow-gpu==1.5.0
#指定安装包的版本

conda deactivate
#退出整个conda环境
