    Matlab_傅里叶变换和巴特沃斯滤波

    创建时间:2021-11-18 20:09
    阅读:
    1. 快速傅里叶变换（FFT）
      1. 示例
  • 巴特沃斯低通滤波

    • 快速傅里叶变换（FFT）

    # 原始信号
Ts = 1/50;                            #采样频率50Hz
t = 0:Ts:10-Ts;                       #采10秒  
x = sin(2*pi*15*t) + sin(2*pi*20*t);  
plot(t,x)
xlabel('Time (seconds)')
ylabel('Amplitude')

#快速傅里叶变换
y = fft(x);                          
fs = 1/Ts;                           # 采样频率
f = (0:length(y)-1)*fs/length(y);    

#做频谱图
plot(f,abs(y))                       #单边幅值谱
xlabel('Frequency (Hz)')
ylabel('Magnitude')
title('Magnitude')
    示例
    Ts = 1/500;                            #采样频率500Hz
x = x1;             # size(x1)= 12802 * 1
plot(x)
xlabel('Time (seconds)')
ylabel('Amplitude')

#快速傅里叶变换
y = fft(x);                          
fs = 1/Ts;     
P2 = abs(y/length(y));
P1 = P2(1:length(y)/2+1,:);
P1(2:end-1)=2*P1(2:end-1);% 采样频率
f = (0:length(y)/2)*fs/length(y);   
size(f)

#做频谱图
plot(f,P1)                       #单边幅值谱
xlabel('Frequency (Hz)')
ylabel('Magnitude')
title('Magnitude')

    注意： 采样信号频率（此处为500Hz）的1/2是能采出的信号的最高频率。（采样定理）

    巴特沃斯低通滤波

    Fs = 250 ;                 # 采样频率的一半，画出的频谱图的最高频率
wp= 120/Fs;                # 设置通带频率，截止频率
ws= 150/Fs;               # 设置阻带频率
Rp=3;                     # 通带最大衰减
Rs=60;                    # 阻带最小衰减
[N,Wn]=buttord(wp,ws,Rp,Rs,'s');     # 求巴特沃斯滤波器阶数N和截止频率Wn
fprintf('巴特沃斯滤波器 N= %4d\n',N); 
fprintf('巴特沃斯滤波器截止频率 Wn= %4d\n',Wn*250); 
[bb,ab]=butter(N,Wn); 
[q,w]=freqz(bb,ab,512);      # 512只影响作图精度，设置为256也可以

# 查看滤波器的效果  
plot(w*Fs/(2*pi),abs(q));
X=filter(bb,ab,x1); 

#滤波后的信号（时域）
plot(X,'linewidth',0.5);
