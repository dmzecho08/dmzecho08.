    hexo配置问题

    hexo
    创建时间:2021-11-29 12:54
    阅读:
    1. 常用命令
    2. 常见错误及解决方法
    3. hexo windows配置

    常用命令

    hexo new "文件名"    #新建.md文件
hexo g       #生成静态文件
hexo server  #启动服务器。默认情况下，访问网址为： http://localhost:4000/
hexo d       #部署网站

    常见错误及解决方法

    error: Spawn failedat ChildProcess.<anonymous> (/home/xx/myblog/node_modules/...)

    1.删除.deploy_git文件后重新部署

    2.依次执行

    hexo clean 
hexo g
hexo d

    hexo windows配置

    主要步骤参考官网hexo官网
    补充参考hexo配置参考,某些步骤存在下载较慢的问题，需要科学上网。
    再将电脑的.ssh key放到github的SSH keys一栏即可建立链接

    运行hexo d报错
WARN Deprecated config detected: "use_date_for_updated" is deprecated
需要将配置文件_config.yml中的
use_date_for_updated 改为 updated_option
